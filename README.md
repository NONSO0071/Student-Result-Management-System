# Student Result Management System

This project is a simple console-based application developed using Python.

## Project Description
The Student Result Management System collects student scores, calculates the total and average, and displays the result.

## SDLC Explanation
This project follows the Software Development Life Cycle (SDLC):

1. Requirement Analysis
2. System Design
3. Implementation
4. Testing
5. Deployment
6. Maintenance

All names and components used in the design match the implementation.
